## 🎯 **How It Works**

### **Query Flow**
```
User Query → Enhanced AI Chat Service → Intent Detection → Crawler Service Router → Specific Crawler → Data Analysis → AI Processing → Rich Response
```

### **Service Integration**
- **Behavioral Crawler**: Whale activity, holder growth, trading patterns
- **Volume Prioritizer**: Top 500 tokens, trending analysis, volume spikes
- **Launchpad Monitor**: New launches, early whale detection, risk assessment
- **Helius Behavioral**: Real-time transaction analysis, smart money detection
- **Jupiter Smart Crawler**: Tiered processing, quality filtering, market data

### **Response Types**
- **Template Responses**: 90% of queries use pre-built templates (FREE)
- **LLM Responses**: Complex queries use OpenAI (PAID)
- **Crawler Responses**: Real-time data from your crawler services
- **Hybrid Responses**: Combination of multiple approaches

## 🚀 **Getting Started**

### **Option 1: Quick Start (Recommended)**
```bash
cd crawler
./quick-start-enhanced-chat.sh
```

### **Option 2: Manual Setup**
```bash
cd crawler
npm install
# Edit .env.local with your API keys
npm run build
npm start
```

### **Option 3: Demo Only**
```bash
cd crawler
npx tsx enhanced-ai-chat-demo.ts
```
